## "Neurons that fire together, wire together."

This project develops an affordable, open-source robotic hand prosthesis that harnesses Hebbian learning to support neurological recovery. By synchronising intentional motor commands with physical movement of the prosthesis, the system aims to reinforce residual motor pathways and promote cortical reorganisation — facilitating functional recovery in patients with peripheral nerve injuries, stroke sequelae, or incomplete spinal cord injuries.

The device is built around low-cost components (servo actuators, EMG/EEG sensors, single-board microcontrollers) and 3d printed hardware, making it accessible for research and clinical prototyping in resource-limited settings.
